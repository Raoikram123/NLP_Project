# Hate and Love in Poetry (NLP Project)

This project analyzes poetry datasets from Project Gutenberg to explore how the themes of *love* and *hate* are expressed linguistically.

## Tasks Completed
1. Data collection and tokenization (NLTK)
2. Publication year histogram by theme
3. Frequency of love/hate-related words
4. Context analysis with WordClouds
5. Sequential occurrence of sentiment terms
6. Cross-sentiment distance analysis
7. Frequent word similarity
8. Poem-level semantic proximity
9. FastText-based title diversity
10. Lexical diversity (adjective/adverb-to-verb ratio)
